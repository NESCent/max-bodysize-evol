Public Instance methods

add_point(x, y = nil)

Purpose: Add a point to the graph line Input:

  • x (float) - The x value for the new point to add, OR an Array containing both the x and y value
  • y (float) [optional] - The y value for the new point to add. When using the array notation, this parameter is not used

Will accept add_point(x, y) for example: add_point(3, 5) or a point as an array add_point([x, y]) for example: add_point([3, 5])

Output: None (Adds the new point to @data)

get_point(number)

Purpose: Retrieve a particular point from the line

Input: number (integer) - the index of the point to retrieve Output: Array - An Array of size 2 where the first value is the value, and the second is the y value: [x, y]

max()

Purpose: Find the maximum value of all the points in the line

Input: None (Uses @data) Output: Float - The maximum value of all the ponts in the line

min()

Purpose: Find the minimum value of all the points in the line

Input: None (Uses @data) Output: Float - The minimum value of all the ponts in the line

scale_line(max_x_value, max_y_value, scale_size = 1000.0)

Purpose: Scale each point in the line to fit the scale that will be used by the chart

Input:

  • max_x_value (Float) - The largest x value that is going to be graphed
  • max_y_value (Float) - The largest y value that is going to be graphed
  • scale_size (Float) - The scale used as th emaximum size of the graph

Output: None (Each point in @data is scaled)

set_color()

Purpose: Set the color of the graph line Input: None (Uses @@nextcolor, and @@colors) Output: None (Sets the color in @color)

size()

Purpose: return the number of data points in the line Input: None Output: Integer - The number of data points in the line
